PROMETHEUS :

Prometheus is a Time series DB , which stores metrics at certain time interval and stores in mumerical format .

It works on a Pull mechanism , instead of servers sending this , it pulls data from the servers via HTTP endpoints .

It uses metrics collectors like node exporters and kube-state-metrics agents to collects the data and conveert to a palin redable format and export them to prometheus via the HTTP API .

│  (Central Server)│ <──────────────────────────────────────────── │   Port :9100      │
└──────────────────┘             Plain-Text Metrics Data           └───────────────────┘

Built-in Alertmanager: Evaluates user-defined rules and routes triggered alerts to Slack, PagerDuty, or email when things break.
