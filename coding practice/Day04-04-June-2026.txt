The below snipped is written IN HCL script of Terrafrom . 
The Coud Provider conf : usues the AWS Plogin and conencts to AWS and selects the region as us-east-1

The next module is used for creation of SSH key pair , showing the encrypted public key tht would be sued to ssh into 

the 3rd module is for the EC2 isntance creation and its specific details like AMI tyoe , size , key_name ltags etc



provider "aws" {
  region = "us-east-1"
}

  key_name = "Pathnex"
  public_key = "ssh-rsa AAA....."
 }

  Name = "Pathnex-EC2"

4. JENKINS       : this module is fo a Jenkins FIle used to run two JOB Stages Build and Test in parallel

pipeline {
    agent any
    stages {
        stage('Parallel Tasks') {
            parallel {
                stage('Build') {
                    steps {
                        sh 'mvn clean compile'
                    }
                }
                stage('Test') {
                    steps {
                        sh 'mvn test'
                    }
                }
            }
        }
    }
}

5. GITLAB CICD   :  This shows a GITLAB CICD SCRIPT , its written in the gitlab-ci.yaml , the script has one stage called BUILD_TEST .
this stage has two jobs MAVEN-BUILD   & MAVEN-TEST . Its using the maven:3.8.1-jdk-17 Docker image . in build its cloning a github epo to execute and in test its tying to go to the sample-java-app dir and test it .



stages:
  - build-test

maven-build:
  stage: build-test
  image: maven:3.8.1-jdk-17
  script:
    - git clone https://github.com/Pathnex/sample-java-app.git
    - cd sample-java-app
    - mvn clean compile
  parallel: 1

maven-test:
  stage: build-test
  image: maven:3.8.1-jdk-17
  script:
    - cd sample-java-app
    - mvn test
  parallel: 1
