MODULE 5 : Gitlab CICD


here the build and test block was running independendly , so the cd sample-java-app was not there for test to use , since build has not prepared it . hence added to make this happen one after other

An artifact section was added so  the repo created in build is carried over to the test part as well .




stages:
  - build
  - test

build:
  stage: build
  image: maven:3.8.1-jdk-17
  script:
    - git clone https://github.com/Pathnex/sample-java-app.git
    - cd sample-java-app
    - mvn clean compile
  artifacts:
    paths:
      - sample-java-app/

test:
  stage: test
  image: maven:3.8.1-jdk-17
  script:
    - cd sample-java-app
    - mvn test
